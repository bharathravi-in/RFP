## Summary

### Key Concepts

1. **Knowledge Profiles**: Dimensional filtering for smart, context-aware knowledge
2. **Knowledge Folders**: Hierarchical organization for discovery and browsing
3. **Knowledge Items**: Atomic content pieces used by both systems
4. **Project Dimensions**: Metadata that drives profile recommendations
5. **Section Types**: Templates with icons, colors, and custom editors

### Design Principles

- **Dual Organization**: Support both filtering and browsing
- **Smart Recommendations**: Auto-match profiles to project dimensions
- **Visual Differentiation**: Each section type has unique appearance and interaction
- **Reusable Content**: Items can be in both profiles and folders
- **Context-Aware**: Knowledge filtered by project dimensions + section needs
