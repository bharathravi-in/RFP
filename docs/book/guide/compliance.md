# Compliance

## Overview
Compliance checking ensures all RFP requirements are addressed.

## How It Works
1. AI extracts requirements from RFP
2. Maps requirements to proposal sections
3. Calculates coverage percentage
4. Identifies gaps

## Viewing Compliance

### 1. Go to Compliance Tab
Click "Compliance" in the toolbar.

### 2. View Matrix
See requirement coverage status.

### 3. Address Gaps
Click on gaps to navigate to sections.

## Status Indicators
| Status | Meaning |
|--------|---------|
| ✅ Covered | Requirement addressed |
| ⚠️ Partial | Needs more detail |
| ❌ Missing | Not addressed |

## Tips
- Check compliance before export
- Address all gaps for higher scores
- Use AI to fill gaps quickly
