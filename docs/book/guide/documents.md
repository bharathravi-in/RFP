# Document Upload

## Supported Formats
- PDF (.pdf)
- Word (.docx)
- Excel (.xlsx)
- PowerPoint (.pptx)

## Steps

### 1. Open Project
Navigate to your project.

### 2. Click Upload
Click "Upload Documents" button.

### 3. Select Files
Drag & drop or browse to select files.

### 4. Wait for Processing
AI analyzes documents (10-30 seconds).

## What Happens

1. **Text Extraction** - Content is parsed
2. **Chunking** - Split into sections
3. **Embedding** - Vector representations created
4. **Storage** - Saved to knowledge base

## Tips
- Upload RFP documents first
- Add company documents for context
- Larger files take longer to process
