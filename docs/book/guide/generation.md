# Content Generation

## How It Works
AI generates content using RAG pipeline:
1. Retrieves relevant context from knowledge base
2. Combines with RFP requirements
3. Generates tailored content

## Steps

### 1. Select Section
Click on a section from the sidebar.

### 2. Click Generate
Click the "Generate" button.

### 3. Wait for AI
Generation takes 15-30 seconds.

### 4. Review & Edit
Review AI-generated content and make edits.

## AI Assistant
Use the AI Assistant for refinements:
- "Make this more concise"
- "Add specific metrics"
- "Change to executive tone"

## Tips
- Upload relevant knowledge docs first
- Review and verify AI-generated content
- Use AI Assistant for quick refinements
