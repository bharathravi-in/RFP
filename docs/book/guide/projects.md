# Creating Projects

## Overview
Projects are the main workspace for managing RFP responses.

## Steps

### 1. Access Dashboard
Navigate to the RFP Pro dashboard after login.

### 2. Click "New Project"
Find the "New Project" button in the top right.

### 3. Fill Details
| Field | Description |
|-------|-------------|
| Name | Project name (e.g., "Government IT RFP") |
| Description | Brief description of the RFP |

### 4. Create
Click "Create" to initialize the project.

## Next Steps
After creating a project:
1. Upload RFP documents
2. Add knowledge base documents
3. Start generating content
